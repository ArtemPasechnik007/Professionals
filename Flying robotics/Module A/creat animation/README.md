# Модуль анимации
Анимация Шоу дронов в целом ничем не отличается от обычной анимации в Blender 3D
План выполнения анимации:
1. Создание окружения 
2. Добавление объектов управления
3. Движение дронов
4. Настройка рендера
5. Запуск самого ренедера
6. Сохранине файлов под нужными названиями
