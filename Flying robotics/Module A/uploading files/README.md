#Выгрузка файлов из анимации 
В КЗ помимо рендера анимации и самого файла анимации от нас требуется выгрузка файлов csv 

? Зачем нужны CSV-файлы в шоу дронов?

CSV (Comma-Separated Values) — это простой и универсальный текстовый формат, который идеально подходит для передачи большого объема данных между разными программами. В контексте шоу дронов они служат "цифровой партитурой" или "сценарием полета".

* Художник/Аниматор работает в 3D-программе (Blender, Maya, After Effects) и думает о визуальной картине.
* Инженер/Оператор работает с программным обеспечением для управления флотом дронов (RotorHazard, UgCS, собственные SDK от производителей вроде Intel, DJI, Skydio).
CSV — это мост между ними. 3D-программа экспортирует данные о движении в CSV, а ПО управления дронами импортирует и исполняет их.

Содержание данных: В каждом CSV-файле обычно содержится таблица, где:

Строки = Кадры (frames) или моменты времени (time steps). Например, каждая строка — это положение дронов на 0.1 секунде.
Столбцы = Параметры для каждого дрона. Минимальный набор: координаты в пространстве (X, Y, Z).
