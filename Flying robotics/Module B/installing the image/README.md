# Установка образа на Raspberry Pi

## Предварительные требования
- Микро-SD карта объемом **16-32 ГБ**
- Картридер
- Компьютер с доступом в интернет
- Программа **balenaEtcher**

## Шаги установки

### 1. Скачивание образа
- Загрузите ZIP-файл с образом системы для Raspberry Pi [образ 23 версии для шоу](https://github.com/CopterExpress/clover/releases/download/v0.23/clover_v0.23.img.zip)
- Распакуйте архив (при необходимости) для получения файла образа (обычно с расширением `.img`)

### 2. Установка balenaEtcher
- Скачайте и установите [balenaEtcher](https://disk.yandex.ru/d/cS4ZXEPOYzpiwA)
- **Важно**: Если на вашем устройстве несколько учетных записей пользователей, запускайте программу от имени администратора

### 3. Запись образа на SD-карту

1. Запустите balenaEtcher
2. Нажмите кнопку **Flash from file**
3. Выберите скачанный файл образа
4. Нажмите **Select target** и выберите вашу микро-SD карту
5. Запустите процесс записи, нажав кнопку **Flash**

### 4. Завершение установки
- Дождитесь окончания процесса записи (программа уведомит о завершении)
- Безопасно извлеките SD-карту из компьютера
- Вставьте карту в Raspberry Pi
- Включите устройство
  


## Примечания
- Весь процесс занимает от 10 до 30 минут в зависимости от скорости карты и размера образа
- После первого запуска Raspberry Pi может потребоваться дополнительное время для настройки системы
