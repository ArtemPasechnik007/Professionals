# Некоторые обозначения и протоколы портов:

SERIAL0 — USB;

SERIAL1 — USART1 (RX/SBUS);

SERIAL2 — USART2 (GPS, с поддержкой DMA);

SERIAL3 — USART3 (пользовательский, TX с возможностью DMA, на разъёме DJI AIR);

SERIAL4 — UART4 (пользовательский, TX с возможностью DMA);

SERIAL5 — UART5 (недоступный, не выведенный);

SERIAL6 — USART6 (только RX, ESC Telem, с возможностью DMA).

Для подключения устройств, таких как модули GPS, радиостанции телеметрии или приёмники, используются порты UART (RX/TX). Для датчиков I2C — контакты SCL и SDA. Провода сигналов ESC для управления мотором подключаются к контактам M1–M4.
