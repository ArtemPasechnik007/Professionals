# Установите ориентацию
ПОДСКАЗКА
Вы можете пропустить этот раздел, если контроллер полёта и компас установлены на транспортном средстве вертикально и направлены вперёд (это ориентация по умолчанию — ROTATION_NONE).

Если автопилот/компас установлены каким-либо другим способом, вам нужно будет указать их ориентацию в виде смещений по осям YAW, PITCH и/или ROLL относительно ориентации «вперёд-вверх» (вращение по часовой стрелке вокруг осей Z, Y и X соответственно).

<img width="1024" height="361" alt="image" src="https://github.com/user-attachments/assets/382f6f5c-7023-43a8-95f2-c36dfa44873d" />

Например, изображение ниже имеет следующие ориентации: ROTATION_NONE, ROTATION_YAW_90,ROTATION_YAW_180,ROTATION_YAW_270.

<img width="1024" height="298" alt="image" src="https://github.com/user-attachments/assets/a794fa06-f841-4d59-b77a-ec3614aacfe3" />


Чтобы задать ориентацию (ориентации) в PX4:

1. Нажмите кнопку Установить ориентацию.

<img width="1044" height="645" alt="image" src="https://github.com/user-attachments/assets/fbbe0d5c-f994-4f93-8459-f8ad4b8e1e62" />

3. Выберите Ориентацию автопилота.
4. Выберите Ориентацию внешнего компаса (эта опция будет отображаться только при наличии внешнего компаса).
5. Нажмите ОК.
