# Включить датчики воздушной скорости
Драйверы датчиков воздушной скорости не запускаются автоматически. Включите каждый тип с помощью соответствующего параметра:

- Sensirion SDP3X: SENS_EN_SDP3X
-- TE MS4525: SENS_EN_MS4525DO
- TE MS5525: SENS_EN_MS5525DS
Вам также следует проверить, что ASPD_PRIMARY равно 1 (см. следующий раздел — это значение по умолчанию).


## Выполнение калибровки
Для калибровки датчика воздушной скорости:

1. Запустите QGroundControl и подключите автомобиль.
2. Включите датчики воздушной скорости, если это еще не сделано (как в предупреждении выше).
3. Выберите значок "Q" > Настройка автомобиля> Датчики (боковая панель), чтобы открыть Настройку датчика.
4. Нажмите кнопку с датчиком скорости полета.
 <img width="1053" height="377" alt="image" src="https://github.com/user-attachments/assets/92bde54d-ecf6-44b9-a79b-a9b49123ac91" />

6. Защитите датчик от ветра (например, обхватите его рукой). Следите за тем, чтобы не перекрыть ни одно из его отверстий.
7. Нажмите OK, чтобы начать калибровку.
8. После запроса подуйте в наконечник трубки Пито, чтобы подать сигнал об окончании калибровки.
ПОДСКАЗКА
Дутьё в трубку также является базовой проверкой правильности установки динамического и статического портов. 
Если они установлены неправильно, то при дутьё в трубку датчик будет показывать большой отрицательный перепад давления, и калибровка прервётся с ошибкой.
9. QGroundControl сообщит вам, прошла ли калибровка успешно.
